# Video Game Sales Analysis Dashboard

## Overview

This project explores global video game sales trends across genres, platforms, and regions using an interactive Tableau dashboard.

## Objectives

* Analyze sales trends over time
* Identify top-performing genres and platforms
* Compare regional sales patterns

## Key Insights

* Action and Sports dominate global sales
* Industry peak observed between 2005–2010
* Post-2011 decline indicates market transition
* PS2 and X360 are top-performing platforms
* North America leads global sales, followed by Europe

## Features

* Interactive year range filter
* Multi-dimensional analysis (genre, platform, region)
* Comparative time-series visualizations

## Tools Used

* Tableau

## How to Use

1. Download the `.twb` file
2. Open in Tableau Desktop
3. Use filters to explore the data
